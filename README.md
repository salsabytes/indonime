## 🚀 Getting Started

### Prerequisites
- Python
- Git

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/salsa-ram/indonime
   cd indonime
   ```
2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   pip install -e .
   ```
3. **Build extensions**
   ```bash
   python setup.py build_ext --inplace
   ```
