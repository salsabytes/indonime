# Security Policy

## Versi yang Didukung

Cuma fokus ke rilisan yang paling baru ya. Kalau ada bug di versi lama, mending langsung update ke yang terakhir.

| Version  | Supported          |
| -------- | ------------------ |
| Latest   | :white_check_mark: |
| < Latest | :x:                |

## Cara Lapor Celah Keamanan

Kalau kamu nemu masalah keamanan yang serius:
1. Buka Issue baru di GitHub repository ini.
2. Jelasin masalahnya secara singkat.
3. Mohon bersabar kalau responnya agak lama.
